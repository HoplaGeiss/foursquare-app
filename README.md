# FoursquareApp - [http://www.muller.tech/foursquare-app/](http://www.muller.tech/foursquare-app/)

## Description

FoursquareApp is an application I built to upskill in angular 2. It uses Foursquare to pull the venue close to the location the user inputs.

## Getting started

- Development server: `ng serve`
- Run the unit tests: `ng test`
- Run the e2e tests: `ng e2e`
- Run the linters: `ng lint`
